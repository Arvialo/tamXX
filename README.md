# tamcat
What is tamcat ?
TamCat is a python reverse shell to help in ctf
# REVERSE SHELL PYTHON
*by arvialo*

**SETUP TAMCAT**
```bash
git clone https://github.com/Arvialo/tamXX
cd tamXX
pip3 install -r requirements.txt
chmod +x tamcat.py
```

**USE TAMCAT**
```bash
./tamcat.py --help
```
